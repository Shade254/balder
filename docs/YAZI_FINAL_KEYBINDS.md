## ğŸ’¡ COMMON WORKFLOWS

### Copy Files
```
1. Navigate to file
2. Press y (yank)
3. Navigate to destination
4. Press p (paste)
```

### Copy Path
```
1. Navigate to file
2. Press w
3. Path is in clipboard
4. Paste anywhere with Ctrl+Shift+V
```

### Open in Cursor
```
1. Navigate to file
2. Press c
3. Cursor opens the file
```

### Quick Edit
```
1. Navigate to file
2. Press n (opens nano)
3. Edit and save (Ctrl+O, Enter, Ctrl+X)
```

### Search Files
```
1. Press /
2. Type search term
3. Navigate results
```

### Create File/Folder
```
For file:
1. Press a
2. Type: myfile.txt
3. Press Enter

For folder:
1. Press a
2. Type: myfolder/
3. Press Enter (note the trailing /)
```

### Toggle Hidden Files
```
Press i to show/hide dotfiles
```
